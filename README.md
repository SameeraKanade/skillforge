# 🚀 SkillForge – Smart Learning & Assessment Platform

## 📌 Project Overview

SkillForge is a full-stack web application designed to help students prepare for placements through quizzes, study materials, and performance tracking.

It provides a structured learning experience with organized content and real-time progress monitoring.

This project was developed as part of the **Infosys Springboard Internship Program**.

---

## ✨ Features

### 🎯 Student Features

* Attempt quizzes across multiple categories
* View performance and results
* Access curated study materials (YouTube integration)

### 🛠️ Admin Features

* Add and manage questions
* Upload study materials (YouTube videos)
* Monitor user performance

### 📊 System Features

* REST API-based architecture
* Responsive UI (mobile + desktop)
* Real-time data updates

---

## 🏗️ Tech Stack

### Frontend

* React (TypeScript)
* Tailwind CSS
* Vite

### Backend

* Spring Boot (Java)
* REST APIs

### 🗄️ Database

* MySQL (Relational Database)
* Designed normalized schema for users, questions, results, and study materials

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```bash
git clone https://github.com/SameeraKanade/skillforge.git
cd skillforge
```

---

### 2️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

### 3️⃣ Backend Setup

```bash
cd backend/backend
./mvnw spring-boot:run
```
